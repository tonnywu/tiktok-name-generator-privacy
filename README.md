# GitHub Pages Deployment

このフォルダの内容をGitHubにアップロードして、GitHub Pagesを有効にするとオンラインでアクセスできます。

Upload the contents of this folder to GitHub and enable GitHub Pages for online access.

## アクセスURL / Access URL

デプロイ後に以下のURLになります / After deployment:
- **https://[username].github.io/[repo-name]/**

## 必要なファイル / Required Files

- ✅ index.html -  홈페이지
- ✅ privacy-policy.html - 개인정보 处理方针
- ✅ terms-of-service.html - 服务条款
- ✅ app-logo.png -  应用图标
